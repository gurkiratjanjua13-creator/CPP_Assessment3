# Restaurant Invoice System (C++)

This is my Assessment 3 project for a simple restaurant invoice system in C++. It lets a user create invoices with multiple items and keeps all invoices saved in a file. [file:1]

## Features

- Login system with roles (admin / normal user) [file:1]  
- Create invoices with item name, quantity and price  
- View all invoices, view by ID and search by date  
- Sort invoices by total amount and see recent invoices  
- Admin can delete invoices, normal user cannot  
- Basic encryption for saving invoices and users in files [file:1]

## How to run

- Compile: `g++ mlb0000715_Assessment3.cpp -o invoice`  
- Run: `./invoice`
